🦁 ¡Aprende Balonmano! — Infografía PWA Interactiva
Una aplicación web progresiva (PWA) educativa e interactiva diseñada para enseñar los fundamentos del balonmano a niños de 8 a 12 años. Construida como un único archivo HTML autocontenido, sin dependencias de servidor ni archivos externos.
🎯 Propósito Pedagógico
•	Transmite una sola idea central: el objetivo del balonmano es marcar goles.
•	Presenta solo 3 a 5 datos clave: 7 jugadores, 3 pasos, 30 minutos y zona de 6m.
•	Usa narrativa y personaje guía mediante un león animado.
•	Mantiene una estructura visual lineal y fácil de explorar.
•	Elimina elementos decorativos innecesarios aplicando el principio de coherencia.
✨ Funcionalidades Principales
•	Mascota animada con movimientos interactivos.
•	Flip cards con efecto 3D.
•	Campo de balonmano en SVG interactivo.
•	Quiz dinámico con retroalimentación inmediata.
•	Aplicación instalable tipo PWA.
•	Accesibilidad mediante etiquetas ARIA.
•	Diseño responsivo para móvil, tablet y escritorio.
🛠️ Tecnologías Utilizadas
•	HTML5
•	CSS3
•	JavaScript Vanilla
•	SVG
•	Service Worker
•	Google Fonts
🚀 Cómo Usar
•	Descarga el archivo index.html.
•	Ábrelo en cualquier navegador moderno.
•	La aplicación funcionará automáticamente.
🎨 Diseño Visual
•	Tipografía: Baloo 2 y Nunito.
•	Colores: naranja, amarillo, azul, verde y rojo.
•	Fondo cálido y limpio.
•	Estilo flat/cartoon.
•	Diseño responsive mobile first.
